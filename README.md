# Love Card Generator

This is a simple Python program that generates love cards with random messages and randomly selected background images from a specified folder. The user can view the generated card and decide whether to save it.
Remember to translate the messages to your preferred language. 

## Features

- Random selection of a message from a predefined list.
- Random selection of a background image from a specified folder.
- Display of the generated card in a new window.
- Option to save the generated card as a PNG image.

## Requirements

- Python 3.x
- Libraries:
  - `tkinter`
  - `Pillow`
  - `random`
  - `os`
  - `glob`
